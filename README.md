This repository contains a Power BI report (SHIVAPRIYA_WORKSHOP_STORE_ANALYTICS.pbix) developed as part of a workshop project. The report focuses on Store Analytics, providing insights into sales, customers, products, and performance trends to support data-driven retail decision-making.

🗂️ Datasets Used
The dashboard uses store transaction and performance data, including:

Sales Data – Total revenue, orders, and average transaction value

Customer Data – Demographics, purchase patterns, and loyalty metrics

Product Data – Category-wise sales, best-sellers, underperforming items

Time Data – Yearly, monthly, and seasonal performance

📊 Key Visualizations
The Power BI report includes:

KPI Cards – Sales, profit, revenue growth, order volume

Line Charts – Monthly/quarterly sales & profit trends

Bar & Column Charts – Category, region, or store-wise comparisons

Pie/Donut Charts – Product/category contribution to revenue

Maps – Regional store performance (if available)

Slicers & Filters – Interactive analysis by time, region, category, etc.

⚙️ Features & Insights
✔️ Sales Trends – Identify peak and low-performing periods
✔️ Customer Analysis – Track customer preferences & behavior
✔️ Product Performance – Highlight best-selling and underperforming products
✔️ Profitability Insights – Margin analysis across categories
✔️ Interactive Dashboards – Drill-down & filter options for detailed insights

🚀 How to Use
Download this repository or the SHIVAPRIYA_WORKSHOP_STORE_ANALYTICS.pbix file.

Open it using Power BI Desktop.

Explore the dashboard using filters, slicers, and visuals.

Use the insights to make data-driven business decisions.

📚 Learning Outcomes
Through this project, you will learn:

Data preparation & cleaning in Power Query

Building a data model with relationships

Creating KPIs & custom measures with DAX

Designing retail-focused dashboards in Power BI

Extracting actionable insights from store data
